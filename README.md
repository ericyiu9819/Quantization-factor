# Quantization-factor

## 项目简介

本项目提供一个基于 AkShare 数据源的 A 股多因子分析与预测平台，包含实时行情监控、因子挖掘、多因子策略评分、机器学习预测、股票池推荐等功能。应用采用 Streamlit 构建，支持网页交互，在 macOS 等桌面环境下即可运行。

## 功能概览

- **实时行情**：根据股票代码展示实时股价、涨跌幅与换手率。
- **多因子策略**：结合动量、波动率、成交量、质量与价值等 A 股特色因子，构建多因子评分。
- **机器学习预测**：使用自适应随机森林模型，基于历史数据预测未来收益，并带有奖励 / 惩罚机制自动调整模型参数。
- **可视化**：使用 Plotly 展示预测与实际收益的对比图表。
- **股票池推荐**：根据自定义价格上限与多因子评分，筛选并推荐股票。
- **投资建议**：基于预测结果给出买入、持有、卖出的建议，并可选择长线或短线投资偏好。
